# NEXUS Launcher V2.0 🚀

Otimizador e Gerenciador de jogos focado em performance extrema para Windows 7, 8, 10 e 11.

### 🛠️ Arsenal Tático
- Limpeza de Temporários, Suspensão do Explorer, Matar Telemetria e mais.
- Interface customizável com monitor de CPU/RAM em tempo real.

### 📖 Minha História
O NEXUS nasceu em Manaus/AM, da necessidade de extrair performance de hardware antigo (Acer Ferrari One). É um projeto focado em provar que PC velho ainda tem lenha pra queimar.

### 📦 Como rodar
1. Instale as dependências: `pip install Pillow psutil requests`
2. Execute: `python nexuslauncher.py`

**Projeto 100% Grátis.**
