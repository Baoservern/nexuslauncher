O Arsenal Tático

6 módulos de combate integrados para aniquilar processos inúteis e priorizar o seu jogo.

🧹
Clean Temp
Extermínio completo de arquivos temporários residuais antes do play.
📡
Kill Telemetry
Bloqueia espionagem e updates automáticos do Windows em tempo real.
⚙️
Stop Services
Desativa serviços pesados e desnecessários com um único clique.
⚡
Priority Inversa
Força bruta para o jogo, silêncio para o sistema operacional.
🖥️
Suspender Explorer
Ganha RAM massiva derrubando a Shell do Windows durante o gameplay.
🛡️
Disable Defender
Controle total sobre o antivírus para evitar quedas súbitas de FPS.

Nascido em Manaus

"O NEXUS nasceu no fruto da necessidade real de um estudante que se recusava a aceitar que o hardware antigo era lixo. O projeto começou como uma tentativa de restaurar a performance de um netbook Acer Ferrari One 200 (ZH6)... provando que com código inteligente e otimização bruta, é possível dar vida nova a qualquer PC."

Especificações do Sistema

O NEXUS prova que hardware antigo ainda tem lenha pra queimar.

Processador Dual Core 1.0 GHz (ou superior)
Memória RAM 4GB DDR2 (ou superior)
Sistema Operacional Windows 7 (Otimizado), 8, 10 ou 11
Python 3.8+ (Dependências: Pillow, psutil, requests)
